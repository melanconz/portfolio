Email Organizer Zachary Melancon

Github

Heroku Web App

This application was completed in React as a test for Return Path in order to my ability as a front end developer. I was asked to build an application that took a JSON list of email senders and related information, and match a wire frame using this data.


click to enlarge image

Initial State

The view matches the wireframe given, exactly, and contains four major components the filterable table, the filter header, the email table, and the email rows themselves.

Filtering

The data can be filtered by specifying a folder from the frop down or searching for a sender using the serach bar.

Changing Folders

Change the folder location of a sender, the organize checkbox must be unchecked to allow the drop down on the right hand side of each Email Row to function. The border color on the right of each Email Row is associated with each folder and changes if the folder is changed.
