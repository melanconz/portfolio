School Search Zachary Melancon

Github

Heroku Web App

This is a personal project I began in May 2017 because I am interested in React. It uses or will use React, Redux, Aphrodite, Flex, and more and will eventually have an associated React Native application. It uses data.gov's College Scorecard API to make finding information about colleges simple. At the moment, a user can search every college in the US by state, click on that college and get a brief explanation of it.

In the future, School Search will include graphic data from the wide variety of information provided by College Scorecard, and eventually, user's will be able to sign up and add prospective colleges to their profile, and universities can easily find prospective students who are interested in attending their school.

Choose a State

Initially, users can choose a state to filter the schools they will see in the school list. Eventually, this page with contain a variety of ways for users to begin their search.

School List

When a state is chosen, all colleges and universities in that State are queried through College Scorecard in groups of 20. The chosen state is displayed at the top of the page, the school's name and ownership are displayed, and users are warned if that school is under investigation. Also, the currently displayed schools can be filtered.

List Footer

At the bottom of the list page, is a custom footer that, when clicked makes an API call for the next page of results for that state. When a school is hovered over, a blue border displays so that it is apperant that schools can be clicked.

School Details

When a school is clicked, a simple detail page is displayed that shows a scools web address, state, locale, Carnegie Classification, and admission rate. Later, more graphs and details will be added to this page.
