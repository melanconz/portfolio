Heroku Web App

This application was completed as a test for AlzCare Labs Inc in order to prove my AngularJS/Angular Materials ability. It uses Flickr's public image search API to requests images associated with the enters search tags.


Search Feature

The search feature allows user to search image using any number of comma separated tags. The image display using Angular Material cards with the title of the image at top if the title was supplied when uploaded to Flickr.
