Inventory Management Zachary Melancon

This application was designed for Approved Oil in order to manage receiving, moving, and verifying of inventory, and it was created using Ionic and AngularJS. I was solely responsible for the functionality of this application including but not limited to adding Cordova's barcode scanner feature, creating data services and API method calls to grab data from Apporved Oilâ€™s Inventory Management QuickBase application, building out the best work flow for the three possible processes, and creating a user experience that was highly functional yet easy to use.

Home

Home page allows users to choose between three possible work processes; Receive, Move, or Physical Inventory

Location

When Move or Physical Inventory is clicked or after a Purchase Order is selected during the Receive process, users select a location at which the event is taking place. If inventory is being moved, user will select location from and then location to appears under the location from section to be chosen

Receive POs

When Recieve is chosen, a user is prompted to choose the associated Purchase Order

Search Item

The last step for each process is to search for the item. If form the Recieve process the PO number and location recieve will be displayed. If from the move process, the location from and location to will be displayed. If from the Physical Inventory page, the location of which the invetory check is happening will be displayed as well as a list of all inventory in that location to be reviewed and checked.

Item List

An items barcode can simply be scanned, but if that's not possible, a user can choose to search for an item from a filterable list. If the item a user chooses or scans is an asset rather than a consummable. The user is then prompted to scan or enter the serial number for that asset.

Save

When Recieve or Move Items are specified or all Physical Invetory has been reviewed for a location, a save button appears and a user can update the associated QuickBase database with the new information.

                (function() {
                  'use strict';

                angular.module('inventorymanagement')
                  .controller('PosCtrl', PosCtrl);

                  function PosCtrl($scope, QB, $ionicScrollDelegate, $state, $stateParams, $timeout, Data, Utils) {

                    init();

                    $scope.reload = reload;
                    $scope.selectPO = selectPO;
                    $scope.refreshPurchaseOrders = refreshPurchaseOrders;
                    $scope.scrollTop = scrollTop;

                    function init() {
                      console.log('Pos controller initialized');
                      loadPOs();
                    }

                    function loadPOs(reload) {
                      $scope.pos = Data.pos.all({reload: reload});
                      $timeout(function() {
                        $scope.loaded = true;
                        Utils.scrollTop();
                      }, 300);
                    }

                    function reload() {
                      loadPOs(true);
                    }

                    function selectPO(po) {
                      $state.go('location', {
                        action: 'receive',
                        po: po.poNumber
                      });
                    }

                    function refreshPurchaseOrders() {
                      loadPOs(true);
                      $scope.$broadcast('scroll.refreshComplete');
                    }

                    function scrollTop() {
                      Utils.scrollTop();
                    };

                  }

                })();
